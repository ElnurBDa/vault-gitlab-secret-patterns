```bash
# Build
docker build -t env-debug .

# Run with .env file
docker run --env-file .env -p 80:80 env-debug

# Or inline
docker run -e USERNAME=admin -e PASSWORD=secret -p 80:80 env-debug
```

`.env`
```env
USERNAME=admin
PASSWORD=secret123
```
